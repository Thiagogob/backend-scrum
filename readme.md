# 🏫 Sistema Web de Reserva de Salas e Laboratórios

[![Scrum Methodology](https://img.shields.io/badge/Methodology-Scrum-blue.svg)](https://www.scrum.org/)
![Status](https://img.shields.io/badge/Status-Em%20Desenvolvimento-green)

Projeto desenvolvido para facilitar a gestão e o agendamento de espaços físicos (salas e laboratórios) em ambiente universitário, otimizando a ocupação e evitando conflitos de horários.

## 📌 Funcionalidades Principais

O sistema foi concebido para atender aos seguintes requisitos básicos:

* **Cadastro de salas:** Registro de novos espaços e laboratórios.
* **Listagem de salas disponíveis:** Consulta rápida de espaços livres para uso.
* **Reserva de sala:** Sistema de agendamento por data e horário.
* **Visualização das reservas:** Painel de controle para consulta de horários ocupados.
* **Cancelamento de reserva:** Opção para liberar horários agendados anteriormente.
